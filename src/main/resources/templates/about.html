О сервисе EasyLink

EasyLink — это сервис для сокращения длинных интернет-ссылок. Превращайте громоздкие URL в короткие и удобные для sharing.

Как это работает

  1. Зарегистрируйтесь и войдите в аккаунт.
  2. Вставьте длинную ссылку на главной странице и нажмите «Сократить».
  3. Получите короткую ссылку фиксированной длины.
  4. При переходе по короткой ссылке вы будете автоматически перенаправлены на исходный ресурс.

Технологии

Серверная часть написана на Java с использованием Spring Framework. Данные хранятся в PostgreSQL. Клиент и сервер обмениваются данными в формате JSON через REST API. Интерфейс работает асинхронно — запросы не блокируют страницу.
